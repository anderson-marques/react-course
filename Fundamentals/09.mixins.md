# Mixins

- Mixins allow common code to be merged into many components

```js

    var HighlightMixin = {
        componentDidUpdate: function(){
            var node = $(React.findDOMNode()));
            node.slideUp();
            node.slideDown();
        }
    });

    var Component = React.createClass({
        mixins: [HighlightMixin]
    });

```
